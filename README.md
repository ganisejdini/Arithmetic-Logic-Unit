# Arithmetic-Logic-Unit
This 32-bit Arithmetic Logic Unit (ALU) is implemented in Verilog to perform a wide range of arithmetic, logical, and shift operations. The module processes two 32-bit input operands based on a 4-bit operation code, which includes support for standard logic gates such as AND, OR, NOR, NAND, and XOR. For arithmetic tasks, it handles addition and subtraction while providing dedicated signed overflow detection logic. It also features a signed multiplication operation that utilizes a 64-bit internal register to accurately detect when a result exceeds the 32-bit output range. In addition to standard arithmetic, the ALU provides comprehensive shifting capabilities, including logical shift left and right, as well as signed arithmetic shifts. The module includes two primary status flags: a zero flag that asserts whenever the 32-bit result is zero, and an overflow flag that triggers during invalid arithmetic outcomes. The entire unit is designed as a combinational block with a default case to ensure a stable output of zero for any undefined operation codes.
